# Java Vinyl Store Management System

A desktop or console-based Java application designed to manage an inventory of vinyl records. It implements full CRUD logic to catalog and maintain a music collection.

## Core Features
* **View Inventory (Read)**: Display a complete list of available vinyls with details (Title, Artist, Release Year, Genre, Price).
* **Add Vinyl (Create)**: Insert new records into the catalog with data validation.
* **Modify Details (Update)**: Edit existing records (e.g., updating stock or price).
* **Remove Records (Delete)**: Permanently delete vinyls from the system.
